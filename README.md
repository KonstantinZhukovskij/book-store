## How to install ?

* **Clone the repo**
   
  ```
  git clone https://github.com/KonstantinZhukovskij/book-store.git
  ```

* **Enter the folder**
  ```
  cd client
  ```
* **Install dependencies**
  ```
  npm install
  ```
* **Run project**
  ```
  npm start
  ```
* **Visit**

    http://localhost:3000
